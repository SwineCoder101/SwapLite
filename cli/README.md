# SwapLite CLI

A command-line interface for interacting with SwapLite.

## Installation

```bash
# Install dependencies
npm install

# Build the project
npm run build
```

## Development

```bash
# Run in development mode
npm run dev

# Run tests
npm test
```

## Usage

The CLI provides the following commands:

### Swap Tokens
```bash
swaplite swap --from <token> --to <token> --amount <amount>
```

### Check Balance
```bash
swaplite balance --token <token>
```
